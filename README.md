# Portfolio Website

A modern, responsive portfolio website built with Next.js, Tailwind CSS, and Shadcn/ui.

## Features

- Full splash page with two-column layout
- Left column: Full-width image display
- Right column: Introductory greeting section
- Responsive design that works on all devices
- Built with Next.js 14 (App Router)
- Styled with Tailwind CSS
- UI components from Shadcn/ui

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - Reusable React components (including Shadcn/ui components)
- `lib/` - Utility functions
- `public/` - Static assets (images, etc.)

## Adding Your Image

Replace the placeholder image at `public/placeholder-image.jpg` with your own image. The image will automatically be displayed in the left column of the landing page.

## Customization

- Update the greeting text in `app/page.tsx`
- Modify colors and styling in `app/globals.css`
- Add more sections to the landing page as needed
